Performance info_outline

Number of items processed in each batch during list refreshes. Smaller batches provide more frequent progress updates on the Status page, while larger batches improve processing efficiency. Default: 300. Recommended: 200-500 for libraries with 1,000-20,000 items.

User Access info_outline

When enabled, regular users can access SmartLists from their home screen sidebar to create and manage their own playlists and collections. Requires the plugins "Plugin Pages" and "File Transformation" to be installed. If disabled or if required plugins are missing, users can only access SmartLists through the admin dashboard. Note: Admins always have access to the user page regardless of this setting.

Cover Images info_outline

Stamps a small badge in the corner of auto-generated and uploaded cover images so smart playlists and collections are easy to spot. Note: when enabled, covers (including uploaded ones) are cropped to Jellyfin's tile proportions (square for playlists, poster/thumb for collections) so the badge stays visible and overview grids stay uniform. When disabled, covers keep their original proportions and uploads are used untouched. Changes are applied on the next refresh of each list.
